>[!summary] Resumen.
>La forma explicita es cuando mostros mismos transformamos el tipo de las variables a través de los diferentes métodos con los que contamos:
>- parseInt()
>- parseFloar()
>- toString()

>[!example] Ejemplo.
```javascript
let numero = 5;
numero.toSrting();
console.log(numero) --> "5"
```
